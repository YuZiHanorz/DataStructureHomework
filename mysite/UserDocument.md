# 系统安装手册

- 如果你是用户：直接打开网页即可

- 如果你是运营商：下载整个文件库，用<pre><code>python3 manage.py runserver</code></pre>命令来运行后端，然后挂机即可

# 用户手册

- 普通用户：请按照提示进行操作即可。

- 管理员：与车次有关的操作全部在你的用户下拉栏里面。
